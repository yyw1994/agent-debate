# Flask API Project

## 环境设置

### 虚拟环境管理

使用 Python 内置的 venv 模块创建虚拟环境：

1. 创建虚拟环境：
```bash
python -m venv venv
```

2. 激活虚拟环境：
```bash
# Mac/Linux
source venv/bin/activate

# Windows
.\venv\Scripts\activate 
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

4. 运行项目：
```bash
python app.py
```

5. 退出虚拟环境：
```bash
deactivate
```

访问 http://localhost:5001/api/hello 查看 API 响应
